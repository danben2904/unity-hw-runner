<h1>Бенца Даниил</h1>

<b>Домашнее задание</b>

Точка входа — сцена <code>main scene</code> (она одна))))

У объекта <code>player</code> находятся почти все характеристики, доступыне для изменения в Инспекторе

<b>Алгоритм</b>

Объект игрока стоит на месте. На него бегут короткие платформы. Как только игрок "убегает" с одной платформы, в конце генериуется ещё одна. Платформы за игроком удаляются. Есть несколько готовых видов платформ с установленными рисунками препятсвий(их 2 вида - высокое и низкое, с разным уроном). Когда генерируется новая платформа, рандомно выбирается один из 10 видов. Движение влево и в право - свободное, есть одиночный прыжок.

<b>Дополнительные функции</b>

Нарастающая скорость игры. Поскольку генерация от скорости не зависит, добавить поддержку это функции было не сложно. Движения игрока(в том числе и прыжок) зависят от скорости.
